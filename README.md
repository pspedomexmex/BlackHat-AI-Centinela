# BlackHat-AI-Centinela
Sistema de auditoría técnica basado en RAG (Retrieval-Augmented Generation) optimizado para hardware local (Ryzen 5 5600G) con enfoque en la integridad de datos y cero alucinaciones.
Nota sobre la bibliografía: El sistema es agnóstico. Los manuales utilizados en mis pruebas pertenecen a sus respectivos autores. Si desea conocer la lista de materiales o cómo obtenerlos legalmente, puede enviarme un mensaje directo.
